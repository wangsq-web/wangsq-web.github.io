使用 hexo 搭建博客, 并尝试使用 github pages 部署。

// 2024-11-21 更新
昨天突然发现这个角落里的项目，之前没时间就荒废了，现在时间倒是多了，可以利用起来了

hexo 命令之类的都忘没了，查官网发现跟新了好几个版本，从5.X 到 7.X，只好重新安装手脚架了，安装版本 hexo-cli@4.3.2
之后就简单了重新初始化一个 hexo 项目 npm init hexo-blog 备份 _config.yml 然后全选复制替换原来的文件就可以了
从官网找到之前的主题 cactus 仙人掌，修改主题配置，然后运行 hexo server 在localhost:4000 就能看到效果了

为了防止以后时间久了忘了怎么使用，特意添加了 README.md，里面详细的命令介绍。
